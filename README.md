# 丧尸塔防 Roguelite

一款融合方格地图塔防、Roguelite 随机构筑、装备组合与疯狂发明主题的轻度策略游戏。

玩家需要在丧尸潮中开拓地图、建造并升级防御塔，通过老虎机获取装备，将装备安装到不同防御塔上，构筑具有独特攻击方式的防线，保护发明家完成最终发明。

## 游戏特色

- **方格地图塔防**：在有限的地图空间内规划道路、建造防御塔并阻挡丧尸路线。
- **装备驱动构筑**：同一件装备安装到不同防御塔上，会产生不同的强化效果。
- **装备组合系统**：通过两件或三件装备触发通用、专属、元素或隐藏传奇组合。
- **元素反应**：利用火、冰、电、毒、风等元素之间的关系，形成更强的战斗效果。
- **Roguelite 成长**：每局随机获得强化，局外逐步解锁装备与内容。
- **疯狂发明主题**：电池、马桶塞、洗衣机、黑洞发生器、鲨鱼发射器等荒诞装备共同组成战斗体系。
- **轻松黑色幽默**：用夸张的武器、丧尸和发明家故事，营造轻松而有辨识度的生存体验。

## 核心体验

游戏的核心构筑公式：

```text
防御塔类型 × 装备 × 装备组合 × Roguelite 强化 × 元素反应 = 本局构筑
```

装备不是简单地提升数值，而是会改变防御塔的攻击形态。例如，同一件装备安装到机枪塔、喷火塔、导弹塔或狙击塔上，会分别产生不同的攻击效果。玩家需要根据地图路线、敌人类型和当前资源，持续调整防线。

## 单局目标

### 胜利条件

- 完成发明进度。
- 挺过全部计划波次。
- 击败最终 Boss。

### 失败条件

- 发明家生命值归零。
- 核心实验装置被摧毁。

单局目标不只是消灭丧尸，还包括持续推进发明进度。完成波次、击杀精英或 Boss、触发特殊强化和完成幸运事件，都可能推动发明进度。

## 单局流程

1. **准备阶段**：查看地图、敌人信息和可用资源，规划初始路线。
2. **开拓地图**：使用铲子翻开未开拓区域，获得新的建造空间、道路或事件。
3. **建造防线**：消耗金币建造防御塔，安排防御塔覆盖丧尸行进路线。
4. **迎战波次**：防御塔自动攻击，玩家根据战况处理资源、升级和装备配置。
5. **老虎机抽取**：使用资源随机获得装备，装备可能附带额外词缀。
6. **调整构筑**：为防御塔安装装备，寻找装备组合和元素反应。
7. **推进发明**：完成波次、处理精英与 Boss，积累发明进度。
8. **结算成长**：完成本局后获得局外解锁和长期成长内容。

## 地图系统

- 地图采用 **10×10 方格**作为基础战斗区域。
- 初始区域提供基础建造空间和防守路线。
- 未开拓区域需要消耗铲子探索。
- 地图开拓会影响道路、建造位置、资源获取和事件触发。
- 玩家需要在“扩大可用空间”和“保留资源应对波次”之间做出选择。

## 防御塔

基础版本包含四种防御塔：

| 防御塔 | 定位 | 战斗特点 |
| --- | --- | --- |
| 机枪塔 | 高速单体输出 | 攻速快、单发伤害低，适合普通丧尸和快速单位，也容易触发命中类效果 |
| 喷火塔 | 直线范围持续伤害 | 持续灼烧前方区域，适合清理成群敌人 |
| 导弹塔 | 范围爆炸输出 | 攻击频率较低但爆发和范围较高，适合处理密集敌群 |
| 狙击塔 | 远程高伤害输出 | 射程远、单次伤害高，适合处理精英和关键目标 |

防御塔升级主要提升基础数值：

- 攻击力
- 攻击速度
- 攻击范围
- 特殊能力触发概率

基础升级不会改变防御塔的攻击形态，真正改变攻击方式的是装备与装备组合。

## 资源系统

| 资源 | 主要用途 |
| --- | --- |
| 金币 | 建造和升级防御塔 |
| 钻石 | 用于老虎机抽取装备或其他高价值选择 |
| 铲子 | 开拓地图、探索未开放区域 |

资源管理是每局决策的重要组成部分。玩家需要在立即强化防线、抽取装备和开拓地图之间分配有限资源。

## 装备系统

### 装备品质

装备按品质分为一至五星，品质越高通常拥有越强的效果或更独特的攻击机制。装备可来自老虎机、事件、发明解锁和特殊奖励。

### 装备类型

设计文档中的装备主题包括：

- 元素装备
- 机械装备
- 厨房用品
- 生活用品
- 生物装备
- 能源装备
- 实验装备
- 传奇发明

已规划的装备示例包括电池、刀片、弹簧、辣椒、风扇、冰箱、磁铁、马桶塞、电锯、毒液罐、披萨盒、微波炉、火箭推进器、雷电线圈、洗衣机、无人机模块、激光发生器、黑洞发生器、时间机器、鲨鱼发射器和终极橡皮鸭。

### 装备词缀

老虎机获得装备时，装备有概率附带词缀。词缀可以进一步改变装备表现，例如：

- **锋利**：提高攻击力
- **轻量**：提高攻击速度
- **加长**：提高攻击范围
- **精准**：提高暴击率
- **稳定**：降低攻击波动
- **节能**：缩短技能触发冷却
- **穿透**：额外穿透目标
- **爆炸**：命中后产生小范围爆炸
- **连锁**：攻击跳跃至其他目标
- **吸附**：改变目标移动或聚集效果

## 装备组合与元素反应

每座防御塔可以安装两件或三件装备。满足特定条件时，防御塔获得额外的组合技能；装备不会永久合成为一件新装备。

组合类型包括：

- 通用组合
- 防御塔专属组合
- 元素组合
- 隐藏传奇组合

元素组合示例：

- **火 + 毒：毒火**，持续灼烧并附加中毒效果。
- **火 + 风：火焰风暴**，扩大火焰影响范围并形成持续伤害。
- **冰 + 风：暴风雪**，形成减速区域并周期性冻结敌人。

## 丧尸与 Boss

敌人通过不同属性检验玩家的构筑：

- **普通感染者**：基础敌人，用于构成常规压力。
- **快速感染者**：移动速度快、生命值低，容易突破低攻速防线。
- **胖感染者**：生命值高、移动速度慢，考验单体伤害能力。
- **护甲感染者**：减少每次受到的伤害，克制依赖高频低伤害的构筑。
- **变异犬**：体型小、移动速度极快，需要快速响应。
- **尖叫者**：周期性召唤额外丧尸，应优先处理。
- **医疗感染者**：治疗附近丧尸，并可能解除燃烧或中毒效果。

Boss 会围绕特定机制设计，要求玩家在输出、控制、路线和装备组合之间进行综合应对。

## 发明家系统

发明家系统同时承担剧情推进、永久装备解锁、单局胜利进度、新装备展示和长期目标等功能。

每局通过以下行为积累发明进度：

- 完成波次
- 击杀精英怪
- 击杀 Boss
- 获得特殊 Roguelite 强化
- 完成幸运事件

当发明进度完成后，玩家进入最终目标结算，并根据本局表现解锁新的内容。

## Roguelite 成长

### 局内成长

- 每局随机获得强化。
- 强化用于调整防御塔属性、装备表现、资源收益或元素反应。
- 随机事件和奖励使每局构筑具有差异。

### 局外成长

- 解锁新的装备和装备图鉴条目。
- 解锁新的发明内容、敌人和 Boss。
- 记录玩家发现过的组合与传奇效果。
- 通过长期目标推动重复游玩。

## 连击与幸运事件

连击系统用于奖励连续击杀、快速清场和高效处理敌人。幸运事件则为单局提供额外选择、资源或特殊强化，使地图探索和波次推进不只围绕固定战斗流程展开。

## 界面布局建议

### 主战斗界面

- **顶部**：当前波次、发明进度、Boss 提示。
- **左上角**：金币、钻石、铲子。
- **中央**：10×10 地图、防御塔、丧尸路线、发明家和实验装置。
- **右侧**：当前选中防御塔的属性、装备槽、升级和出售操作。
- **底部**：防御塔建造栏及当前可用建造选项。

### 局外界面

建议包含：

- 发明家与剧情推进
- 装备图鉴
- 已发现的装备组合
- 永久解锁与长期成长
- Boss 和敌人图鉴

## 开发重点

建议优先验证以下内容：

1. 10×10 方格地图与丧尸寻路。
2. 四种防御塔的基础攻击循环和目标选择。
3. 建造、升级、出售和资源消耗。
4. 装备安装后对不同防御塔产生差异化效果。
5. 两件装备组合和至少两组元素反应。
6. 波次、敌人属性和失败条件。
7. 发明进度、Boss 触发与单局结算。
8. 局内随机强化和局外解锁的最小闭环。

## 设计原则

- **策略来自选择**：玩家应持续在路线、资源、防御塔和装备之间做取舍。
- **装备改变玩法**：装备优先提供新的攻击方式或战斗行为，而不是单纯堆数值。
- **构筑可读可试验**：组合效果应容易理解，并鼓励玩家尝试不同搭配。
- **每局都有差异**：随机装备、词缀、强化和事件共同保证局内体验变化。
- **幽默服务于系统**：荒诞装备和发明家主题应与真实的战斗效果绑定，形成记忆点。
- **节奏清晰**：准备、战斗、奖励、构筑调整和下一波之间应形成稳定循环。
